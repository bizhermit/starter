# 使い方

## Web Application

### デバッグ起動

```bash
> npm run server
```

### 起動

```bash
> npm run start
```

### 確認

http://localhost:3000

## Desktop Application

### デバッグ起動

```ps
> npm run electron
```

### 実行ファイル確認

```ps
> npm run confirm
```

### インストーラー作成

```ps
> npm run build
```
